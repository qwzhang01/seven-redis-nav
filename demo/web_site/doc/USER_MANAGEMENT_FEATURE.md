# 用户管理功能

## 功能概述

后台管理系统新增了用户管理功能，管理员可以查看、搜索、筛选和管理系统中的所有用户。

## 功能特性

### 1. 用户列表展示
- 以表格形式展示所有用户信息
- 显示用户基本信息：用户名、昵称、邮箱、用户ID
- 显示用户类型：普通用户/管理员
- 显示用户状态：活跃/未激活/已锁定
- 显示注册时间和最后登录时间

### 2. 统计数据
在页面顶部展示四个关键统计指标：
- **总用户数**：系统中所有用户的总数
- **活跃用户**：状态为"活跃"的用户数量
- **今日新增**：当天注册的新用户数量
- **已锁定**：被锁定的用户数量

### 3. 搜索和筛选
- **搜索功能**：支持按用户名、邮箱或昵称进行实时搜索
- **状态筛选**：可按用户状态筛选（全部/活跃/未激活/已锁定）
- **类型筛选**：可按用户类型筛选（全部/普通用户/管理员）

### 4. 用户操作
每个用户行提供以下操作：
- **查看详情**：查看用户的完整信息，包括：
  - 基本信息（用户ID、用户名、昵称、邮箱、手机号、用户类型）
  - 状态信息（账户状态、邮箱验证、手机验证、注册时间、最后登录、创建时间）
- **编辑**：编辑用户信息（待实现）
- **锁定/解锁**：
  - 活跃用户可以被锁定
  - 已锁定用户可以被解锁

### 5. 用户详情弹窗
点击"查看详情"按钮后，会弹出模态窗口显示用户的详细信息：
- 基本信息区域
- 状态信息区域
- 响应式设计，支持移动端查看

## 技术实现

### 文件结构
```
src/
├── views/
│   └── admin/
│       └── AdminUsers.vue          # 用户管理页面
├── router/
│   └── index.ts                    # 路由配置（已添加用户管理路由）
└── views/layout/
    └── AdminLayout.vue             # 管理后台布局（已添加用户管理菜单）
```

### 路由配置
- **路径**：`/admin/users`
- **名称**：`AdminUsers`
- **权限**：需要管理员权限（`requiresAuth: true, requiresAdmin: true`）

### 组件特性
- 使用 Vue 3 Composition API
- 响应式数据管理
- 实时搜索和筛选
- 优雅的UI设计，与现有管理后台风格一致
- 使用 Lucide Vue Next 图标库

### 数据结构
使用 `UserProfile` 类型（来自 `@/utils/userApi`）：
```typescript
interface UserProfile {
  id: string
  username: string
  nickname: string
  email: string
  email_verified: boolean
  phone?: string
  phone_verified: boolean
  avatar_url?: string
  user_type: 'customer' | 'admin'
  registration_time: string
  last_login_time?: string
  status: 'active' | 'inactive' | 'locked'
  create_time: string
  update_time?: string
}
```

## 使用说明

### 访问用户管理
1. 以管理员身份登录系统
2. 进入管理后台（`/admin`）
3. 点击左侧菜单的"用户管理"

### 搜索用户
在搜索框中输入用户名、邮箱或昵称的关键词，列表会实时过滤显示匹配的用户。

### 筛选用户
使用状态和类型下拉菜单进行筛选，可以组合使用多个筛选条件。

### 查看用户详情
点击用户行右侧的"眼睛"图标，查看该用户的完整信息。

### 锁定/解锁用户
- 对于活跃用户，点击"锁定"图标可以锁定该用户
- 对于已锁定用户，点击"解锁"图标可以解锁该用户

## 样式说明

页面使用了与现有管理后台一致的设计风格：
- **玻璃态卡片**：使用 `glass-card` 类
- **深色主题**：基于 `dark-` 系列颜色
- **主题色**：使用 `primary-` 和 `accent-` 系列颜色
- **状态颜色**：
  - 活跃：绿色（`emerald-`）
  - 未激活：琥珀色（`amber-`）
  - 已锁定：红色（`red-`）
  - 管理员：紫色（`purple-`）
  - 普通用户：蓝色（`blue-`）

## 注意事项

1. **权限控制**：只有管理员可以访问用户管理页面
2. **数据安全**：用户密码等敏感信息不会在前端显示
3. **操作确认**：重要操作（如锁定用户）应该添加确认提示
4. **实时更新**：当前为本地状态更新，实际应该调用API并刷新数据
