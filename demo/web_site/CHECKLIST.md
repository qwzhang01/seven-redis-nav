# 生产环境部署检查清单

## 部署前检查 ✓

### 环境准备
- [ ] Docker 版本 >= 20.10
- [ ] Docker Compose 版本 >= 2.0
- [ ] 服务器内存 >= 2GB
- [ ] 服务器磁盘空间 >= 5GB
- [ ] 网络连接正常
- [ ] 防火墙规则已配置

### 配置检查
- [ ] `.env` 文件已创建并配置
- [ ] 端口未被占用（默认 80）
- [ ] 后端 API 服务正常运行
- [ ] `docker/nginx.conf` 配置正确
- [ ] 域名解析已配置（如需要）

### 代码准备
- [ ] 代码已提交到版本控制
- [ ] 依赖包版本已锁定
- [ ] 构建脚本已测试
- [ ] 静态资源已优化

## 部署过程检查 ✓

### 构建阶段
- [ ] 镜像构建成功
- [ ] 构建日志无错误
- [ ] 镜像大小合理（< 200MB）
- [ ] 构建时间可接受（< 5分钟）

### 部署阶段
- [ ] 旧容器已备份
- [ ] 新容器启动成功
- [ ] 健康检查通过
- [ ] 零停机切换完成

### 验证阶段
- [ ] 首页可正常访问
- [ ] 静态资源加载正常
- [ ] API 接口响应正常
- [ ] 路由跳转正常
- [ ] 浏览器控制台无错误

## 部署后检查 ✓

### 功能验证
- [ ] 核心功能正常
- [ ] 用户登录正常
- [ ] 数据展示正常
- [ ] 表单提交正常
- [ ] 文件上传/下载正常

### 性能检查
- [ ] 页面加载时间 < 3秒
- [ ] API 响应时间 < 500ms
- [ ] CPU 使用率 < 80%
- [ ] 内存使用率 < 80%
- [ ] 网络延迟正常

### 安全检查
- [ ] HTTPS 已配置（生产环境）
- [ ] 安全头已设置
- [ ] 敏感信息已隐藏
- [ ] 访问日志正常记录
- [ ] 错误信息不泄露敏感数据

### 监控检查
- [ ] 容器状态监控正常
- [ ] 日志收集正常
- [ ] 告警规则已配置
- [ ] 健康检查端点正常
- [ ] 资源使用监控正常

## 运维准备 ✓

### 文档准备
- [ ] 部署文档已更新
- [ ] 运维手册已准备
- [ ] 故障排查指南已准备
- [ ] 回滚方案已准备

### 备份策略
- [ ] 镜像备份已完成
- [ ] 配置文件已备份
- [ ] 数据库已备份（如有）
- [ ] 备份恢复已测试

### 应急预案
- [ ] 回滚脚本已准备
- [ ] 紧急联系人已确认
- [ ] 故障处理流程已明确
- [ ] 备用方案已准备

## 常见问题检查 ✓

### 容器问题
- [ ] 容器是否正常运行
- [ ] 容器日志是否有错误
- [ ] 容器资源是否充足
- [ ] 容器网络是否正常

### 应用问题
