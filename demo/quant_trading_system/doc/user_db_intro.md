### 数据库设计规范
#### 所有数据库包含5个公共字段
- id 主键 
- createBy 创建人 
- createTime 创建时间
- updateBy 更新人
- updateTime 更新时间
- enableFlag 逻辑删除标记，1正常0删除

#### 字段命名规范
- 所有的外建都是逻辑外建，不加实体的真实外建，在代码中控制
- 所有的表明都用下划线的方式命名
- 所有的字段使用小驼峰
