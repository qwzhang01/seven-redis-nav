## 📚 功能说明

### 1️⃣ 技术指标计算

系统支持以下指标：

| 指标名称 | 说明 | 用途 |
|---------|------|------|
| SMA | 简单移动平均线 | 判断趋势方向 |
| EMA | 指数移动平均线 | 更敏感的趋势指标 |
| MACD | 平滑异同移动平均线 | 判断买卖时机 |
| RSI | 相对强弱指标 | 判断超买超卖 |
| BOLL | 布林带 | 判断价格波动范围 |
| ATR | 真实波动幅度 | 衡量市场波动性 |
| KDJ | 随机指标 | 判断超买超卖 |

**使用示例：**
```bash
# 运行demo.py会自动计算所有指标
python examples/demo.py
```

---

### 2️⃣ 策略回测

**什么是回测？**
用历史数据测试你的交易策略，看看能赚多少钱。

**系统会计算：**
- 📈 总收益率：赚了多少钱（百分比）
- 📉 最大回撤：最大亏损幅度
- 📊 夏普比率：风险调整后的收益（越高越好）
- 🎯 胜率：赚钱的交易占比
- 💰 盈亏比：平均盈利/平均亏损

**运行回测：**
```bash
# 方式1：使用命令行
quant backtest --strategy DualMAStrategy

# 方式2：运行示例程序
python examples/demo.py
```

---

### 3️⃣ AI智能交易

**什么是AI交易？**
使用强化学习算法，让AI自己学习如何交易。

**运行AI训练：**

### 4️⃣ API服务

**启动服务：**
```bash

```

**可用接口：**

| 接口路径 | 功能 |
|---------|------|
| `/api/v1/market` | 获取行情数据 |

| `/api/v1/trading` | 执行交易 |

浏览器打开 http://localhost:8000/docs

## ❓ 常见问题

### Q1: 安装时报错 "ta-lib not found"
**解决方法：**

**Mac系统：**
```bash
brew install ta-lib
pip install ta-lib
```

**Ubuntu/Debian：**
```bash
sudo apt-get install ta-lib
pip install ta-lib
```

**Windows：**
下载预编译包：https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib

---

### Q2: 运行demo.py报错
**检查步骤：**
1. 确认Python版本 >= 3.11
2. 确认已安装依赖：`pip install -e .`
3. 确认在正确目录：`cd examples`

---

### Q3: 如何使用自己的数据？
**修改demo.py中的数据生成部分：**
```python
# 不使用generate_sample_data()
# 改为读取你自己的CSV文件
import pandas as pd
df = pd.read_csv('your_data.csv')
```

---

### Q4: 如何修改策略参数？
**编辑demo.py文件：**
```python
strategy = DualMAStrategy(
    fast_period=5,   # 快速均线周期（改这里）
    slow_period=20,  # 慢速均线周期（改这里）
)
```

---

### Q5: 回测结果不理想怎么办？
**优化建议：**
1. 调整策略参数（均线周期、止损点等）
2. 增加更多技术指标
3. 优化入场和出场条件
4. 使用更长的历史数据测试
