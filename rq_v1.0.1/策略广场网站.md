## 二、首页

### 2.1 顶部导航

| 导航项 | 说明 |
|--------|------|
| 系统策略 | 跳转至系统预设策略列表页 |
| 信号广场 | 跳转至信号广场列表页 |
| 收益排行榜 | 跳转至信号收益排行榜 |
| 我的 | 跳转至用户中心 |

### 2.2 核心模块入口区

- **系统预设策略（卡片）** — 以卡片形式展示，点击进入系统预设策略列表
- **信号广场（卡片）** — 以卡片形式展示，点击进入信号广场列表

### 2.3 热门 / 推荐信号

展示当前热门或系统推荐的信号，方便用户快速发现优质信号。

### 2.4 风险提示 & 合规说明

页面底部展示风险提示与合规声明，确保用户知悉量化交易相关风险。

## 三、系统预设策略

### 3.1 策略列表

#### 筛选条件

| 筛选项 | 可选值 |
|--------|--------|
| 市场类型 | 现货、合约 |
| 策略类型 | 区间、趋势、突破 |
| 风险等级 | 低、中、高（或多级划分） |

#### 策略卡片

每张策略卡片包含以下信息：

| 字段 | 说明 |
|------|------|
| 策略名称 | 策略的显示名称 |
| 简要说明 | 策略的简短描述 |
| 适用市场 | 现货 / 合约 |
| 风险等级 | 该策略的风险等级标识 |
| 查看详情 | 点击进入策略详情页 |

### 3.2 策略详情页

| 模块 | 说明 |
|------|------|
| 策略介绍 | 策略的整体介绍与适用场景 |
| 策略逻辑说明 | 策略的交易逻辑与运作原理 |
| 参数说明 | 策略涉及的可配置参数及其含义 |
| 风险提示 | 使用该策略的风险说明 |
| 启动策略 / 参数配置 | 用户配置参数后启动策略运行 |

### 3.3 策略运行 / 参数配置

用户选择策略后，可自定义参数并启动策略运行，进入实际交易流程。

---

## 四、信号广场

### 4.1 信号列表

#### 筛选条件

| 筛选项 | 说明 |
|--------|------|
| 来源平台 | 信号的来源交易平台 |
| 模拟 / 实盘 | 区分模拟信号与实盘信号 |
| 运行天数 | 按信号运行时长筛选 |
| 风险等级 | 信号的风险等级 |

#### 信号卡片

每张信号卡片包含以下信息：

| 字段 | 说明 |
|------|------|
| 信号名称 | 信号的显示名称 |
| 来源平台 | 信号来源的交易平台 |
| 累计收益率 | 信号自运行以来的累计收益率 |
| 最大回撤 | 信号历史最大回撤幅度 |
| 运行状态 | 当前信号的运行状态 |
| 查看详情 | 点击进入信号详情页 |

### 4.2 信号详情页

#### 信号基本信息

| 字段 | 说明 |
|------|------|
| 来源平台 | 信号来源的交易平台 |
| 是否模拟 | 标识信号为模拟盘或实盘 |
| 运行天数 | 信号已运行的天数 |
| 当前状态 | 信号当前运行状态 |

#### 收益表现

| 指标 | 说明 |
|------|------|
| 累计收益率 | 信号的总收益率 |
| 最大回撤 | 信号的历史最大回撤 |
| 收益曲线 | 以图表形式展示信号的收益变化趋势 |

#### 当前持仓（只读）

展示信号当前的持仓情况，用户仅可查看，不可修改。

#### 跟单设置

| 配置项 | 说明 |
|--------|------|
| 跟单资金 | 用户分配给跟单的资金额度 |
| 跟单比例 | 按信号仓位的比例进行跟单 |
| 启动跟单 | 确认配置后启动跟单 |

### 4.3 跟单配置

用户完成跟单参数设置后，系统根据配置自动跟随信号执行交易。

## 五、信号收益排行榜

### 5.1 排序方式

| 排序项 | 说明 |
|--------|------|
| 累计收益率 | **默认排序**，按信号累计收益率降序 |
| 累计收益 USDT | 按信号累计收益（USDT 计价）降序 |
| 最大回撤 | 按最大回撤升序（回撤越小越靠前） |
| 运行天数 | 按信号运行天数排序 |

### 5.2 排行榜列表

| 字段 | 说明 |
|------|------|
| 排名 | 信号在当前排序下的排名 |
| 信号名称 | 信号的显示名称 |
| 来源平台 | 信号来源的交易平台 |
| 累计收益率 | 信号的累计收益率 |
| 最大回撤 | 信号的最大回撤 |
| 跟随人数 | 当前跟随该信号的用户数量 |
| 查看信号 | 点击进入信号详情页 |

## 六、我的（用户中心）

### 6.1 我的策略

| 分类 | 说明 |
|------|------|
| 运行中 | 当前正在运行的策略列表 |
| 已结束 | 已停止运行的历史策略列表 |

### 6.2 我的信号跟单

| 分类 | 说明 |
|------|------|
| 跟随中 | 当前正在跟随的信号列表 |
| 已停止 | 已停止跟随的历史信号列表 |

### 6.3 收益统计

展示用户的整体收益数据，包括策略收益和信号跟单收益的统计汇总。

### 6.4 API / 账户管理

用户管理交易所 API 密钥绑定及账户相关设置。

## 七、管理后台（内部）

| 模块 | 说明 |
|------|------|

| 排行榜规则配置 | 配置排行榜的排序规则与展示逻辑 |
