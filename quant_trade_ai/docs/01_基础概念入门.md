# AI 量化交易入门指南

## 第一章：基础概念

---

## 1. 什么是量化交易？

**量化交易** (Quantitative Trading) 是指使用数学模型和计算机程序来分析市场数据，自动做出买卖决策的交易方式。

### 传统交易 vs 量化交易

| 对比项 | 传统交易 | 量化交易 |
|-------|---------|---------|
| 决策方式 | 人工分析、主观判断 | 算法自动决策 |
| 情绪影响 | 容易受贪婪/恐惧影响 | 无情绪干扰 |
| 执行速度 | 较慢 | 毫秒级响应 |
| 数据处理 | 有限 | 可处理海量数据 |

### 量化交易的核心流程

```
数据获取 → 特征工程 → 策略开发 → 回测验证 → 实盘交易
   ↓          ↓          ↓          ↓          ↓
历史价格    技术指标    交易信号    模拟盈亏    真实执行
成交量      基本面      买/卖/持有   风险评估    风控监控
```

## 2. 什么是强化学习？

**强化学习** (Reinforcement Learning, RL) 是机器学习的一个分支，让智能体通过与环境交互来学习最优策略。

### 核心概念图解

```
                    ┌─────────────────┐
                    │     环境        │
                    │  (股票市场)      │
                    └────────┬────────┘
                             │
              ┌──────────────┴──────────────┐
              │                             │
              ▼                             │
        ┌─────────┐                         │
        │  状态   │ ◄────────────────────────┤
        │ (State) │    观测市场状态           │
        └────┬────┘                         │
             │                              │
             ▼                              │
        ┌─────────┐                         │
        │  智能体  │                         │
        │ (Agent) │                         │
        └────┬────┘                         │
             │                              │
             ▼                              │
        ┌─────────┐                         │
        │  动作   │ ─────────────────────────┤
        │(Action) │    执行交易              │
        └────┬────┘                         │
             │                              │
             ▼                              │
        ┌─────────┐                         │
        │  奖励   │ ◄────────────────────────┘
        │(Reward) │    盈亏反馈
        └─────────┘
```

### 关键术语解释

| 术语 | 在交易中的含义 | 例子 |
|-----|--------------|------|
| **状态 (State)** | 当前市场信息 | 价格、技术指标、持仓 |
| **动作 (Action)** | 交易决策 | 买入、卖出、持有 |
| **奖励 (Reward)** | 交易结果反馈 | 盈利为正，亏损为负 |
| **策略 (Policy)** | 决策规则 | 根据状态选择动作的函数 |
| **价值 (Value)** | 预期累计收益 | 从当前状态能获得的总收益 |

## 3. 什么是深度学习？

**深度学习** (Deep Learning) 使用多层神经网络来学习数据的复杂模式。

### 神经网络基础结构

```
输入层              隐藏层              输出层
 (状态)            (特征提取)           (Q值)

  ○                  ○                  ○ ← Q(买入)
  ○ ──────────────► ○ ──────────────► ○ ← Q(持有)
  ○                  ○                  ○ ← Q(卖出)
  ○                  ○
  ↑                  ↑                  ↑
技术指标          学习到的模式        每个动作的价值
```

### 常用深度学习概念

| 概念 | 解释 | 作用 |
|-----|------|-----|
| **神经元** | 网络的基本单元 | 接收输入，产生输出 |
| **权重** | 连接的强度 | 决定输入的重要性 |
| **激活函数** | 非线性变换 | 让网络学习复杂模式 |
| **损失函数** | 误差度量 | 衡量预测与实际的差距 |
| **梯度下降** | 优化算法 | 调整权重减小误差 |

## 4. DQN 算法详解

**DQN** (Deep Q-Network) 是将深度学习与 Q-learning 结合的算法。

### Q-Learning 基本思想

Q值表示在某状态下执行某动作的价值：

```
Q(状态, 动作) = 即时奖励 + 折扣因子 × 未来最大Q值
```

用公式表示：
```
Q(s, a) = r + γ × max Q(s', a')
          ↑   ↑       ↑
        奖励 折扣   下一状态的最优动作价值
```

### DQN 的关键技术

1. **经验回放 (Experience Replay)**
   - 将交易经验存储到缓冲区
   - 随机采样进行训练
   - 打破数据相关性，提高学习效率

2. **目标网络 (Target Network)**
   - 使用两个网络：主网络和目标网络
   - 目标网络定期从主网络复制
   - 提高训练稳定性

3. **ε-贪婪策略 (ε-greedy)**
   - 以概率 ε 随机探索
   - 以概率 1-ε 选择最优动作
   - 平衡探索与利用

### DQN 训练流程

```
1. 初始化网络和经验缓冲区
2. 循环每个回合：
   a. 重置环境，获取初始状态
   b. 循环每一步：
      - 根据ε-greedy选择动作
      - 执行动作，获得奖励和新状态
      - 存储经验 (s, a, r, s', done)
      - 从缓冲区采样，计算目标Q值
      - 更新网络参数
      - 定期更新目标网络
3. 返回训练好的模型
```

## 5. 技术指标简介

技术指标是根据历史价格和成交量计算出的数值，帮助判断市场趋势。

### 常用技术指标

| 指标 | 全称 | 用途 | 计算方式 |
|-----|------|-----|---------|
| **RSI** | 相对强弱指数 | 判断超买超卖 | 基于涨跌幅度 |
| **MACD** | 移动平均收敛散度 | 判断趋势方向 | 快慢均线差值 |
| **BB** | 布林带 | 判断波动范围 | 均线±标准差 |
| **SMA** | 简单移动平均 | 平滑价格 | N日价格平均 |
| **ATR** | 平均真实波幅 | 衡量波动性 | 真实波幅均值 |

### 指标解读示例

```
RSI 值解读：
- RSI > 70: 超买，可能回调（考虑卖出）
- RSI < 30: 超卖，可能反弹（考虑买入）
- RSI ≈ 50: 中性，趋势不明

MACD 信号：
- MACD 上穿信号线: 金叉（买入信号）
- MACD 下穿信号线: 死叉（卖出信号）
```
