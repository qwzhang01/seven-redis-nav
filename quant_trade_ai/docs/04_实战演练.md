## 练习2：理解训练过程

让我们逐步理解训练循环：

```python
# 训练循环的核心代码
for episode in range(50):                    # 重复50个回合
    state, _ = train_env.reset()             # 重置环境
    done = False
    
    while not done:                          # 一个回合的循环
        action = agent.select_action(state)  # 选择动作
        next_state, reward, done, _, info = train_env.step(action)  # 执行
        agent.store_experience(...)          # 存储经验
        agent.train_step()                   # 训练网络
        state = next_state                   # 更新状态
```

**关键概念解释：**

| 概念 | 解释 |
|-----|------|
| Episode (回合) | 完整的一轮交易，从开始到结束 |
| State (状态) | 当前市场的信息表示 |
| Action (动作) | 智能体的决策：买/卖/持有 |
| Reward (奖励) | 这一步获得的收益/损失 |
| Done (结束) | 是否到达数据末尾 |

## 练习8：比较不同算法

项目提供了两种算法：DQN 和 PPO。

### 8.1 使用 PPO 训练

创建新文件 `train_ppo.py`：

```python
from trading_env import TradingEnvironment
from ppo_agent import PPOAgent, train_ppo
from quick_demo import generate_synthetic_data

# 生成数据
df = generate_synthetic_data(n_days=500)
train_df = df.iloc[:400]

# 创建环境
env = TradingEnvironment(train_df)

# 创建 PPO 智能体
agent = PPOAgent(
    state_size=14,
    action_size=3,
    learning_rate=0.0003
)

# 训练
rewards, profits = train_ppo(env, agent, n_episodes=50)
```

### 8.2 比较两种算法

| 特点 | DQN | PPO |
|-----|-----|-----|
| 学习方式 | 价值函数 | 策略梯度 |
| 稳定性 | 较低 | 较高 |
| 样本效率 | 较高 | 较低 |
| 适用场景 | 离散动作 | 离散/连续动作 |

## 常见问题

**Q: 训练很慢怎么办？**
- 减少训练回合数
- 使用更小的神经网络
- 如果有 GPU，检查 PyTorch 是否正确使用

**Q: 智能体总是亏钱？**
- 增加训练回合数
- 调整奖励函数
- 检查数据是否合理

**Q: 过拟合怎么办？**
- 使用更多的训练数据
- 增加 Dropout 比例
- 使用数据增强
