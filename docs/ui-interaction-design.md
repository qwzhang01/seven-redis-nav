# Seven Redis Nav - UI 交互设计文档

> 版本：v1.0 | 日期：2026-04-22

---

## 1. 设计理念

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **原生感** | 遵循 macOS Human Interface Guidelines，使用原生控件风格 |
| **高效流** | 信息密度适中，操作路径最短化，键盘优先 |
| **清晰度** | 层级分明，重点突出，数据类型用色彩区分 |
| **安全感** | 危险操作强提醒，不可逆操作二次确认 |

### 1.2 视觉风格

- **风格**：现代简约，毛玻璃效果 + 微投影，接近 SF Symbols 风格
- **配色**：深色模式为主（Redis 经典红色 #DC382D 为品牌色），支持浅色模式
- **圆角**：8px 大卡片 / 6px 小组件 / 4px 按钮
- **字体**：SF Pro Text（系统默认），代码区使用 SF Mono / JetBrains Mono

## 4. 交互流程

### 4.1 首次使用流程

```
启动应用
    │
    ▼
欢迎页（首次引导）
    │
    ├── [快速连接] ──→ 简化连接表单（Host + Port + Password）
    │                       │
    │                       ▼
    │                  连接成功 → 进入主界面
    │                       │
    │                  连接失败 → 显示错误，返回修改
    │
    └── [导入配置] ──→ 选择配置文件 → 解析连接 → 进入主界面
```

### 4.2 数据操作流程

```
选择 Key
    │
    ├── 查看 → 自动识别类型 → 对应查看器渲染
    │
    ├── 编辑 → 进入编辑模式 → 修改内容 → 保存确认 → 写入 Redis
    │                                           │
    │                                      保存成功 → 提示 ✅
    │                                      保存失败 → 提示 ❌ + 错误原因
    │
    ├── 删除 → 确认弹窗 → 输入 Key 名确认 → 删除
    │
    └── 复制 → 复制 Key 名 / 复制 Value / 复制为 JSON
```

### 4.3 危险操作确认流程

```
触发危险操作（FLUSHDB / FLUSHALL / 批量删除）
    │
    ▼
┌──────────────────────────────┐
│  ⚠️ 危险操作                   │
│                               │
│  即将执行 FLUSHDB 操作         │
│  此操作将删除 db0 中所有数据   │
│  且不可恢复！                  │
│                               │
│  请输入 "FLUSHDB" 以确认：    │
│  ┌────────────────────────┐  │
│  │                        │  │
│  └────────────────────────┘  │
│                               │
│      [取消]    [确认执行]     │
└──────────────────────────────┘
```

### 4.4 批量操作流程

```
多选 Key（⌘+点击 / Shift+点击 / 全选 ⌘+A）
    │
    ▼
批量操作工具栏
    ├── 批量删除 → 确认弹窗 → 执行
    ├── 批量设置 TTL → 输入秒数 → 执行
    ├── 批量复制 Key 名 → 剪贴板
    └── 批量导出 → 选择格式 → 下载
```

## 7. 动效设计

| 场景 | 动效 | 时长 |
|------|------|------|
| 侧栏展开/收起 | 宽度渐变 + 内容淡入 | 200ms ease |
| Key 树展开/折叠 | 高度渐变 + 子节点交错淡入 | 150ms ease |
| Tab 切换 | 内容左右滑动 | 200ms ease |
| Toast 出现 | 从底部滑入 + 淡入 | 200ms ease-out |
| Toast 消失 | 向上滑出 + 淡出 | 150ms ease-in |
| 数据加载 | 骨架屏闪烁 | 1.2s 循环 |
| 连接状态变化 | 圆点颜色渐变 | 300ms ease |
| 按钮悬浮 | 微上浮 + 阴影加深 | 150ms ease |
| 删除确认 | 对话框缩放弹出 | 200ms cubic-bezier |
