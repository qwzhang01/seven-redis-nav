> 版本：v1.0 | 日期：2026-04-22

---

## 1. 产品概述

### 1.1 产品名称

**Seven Redis Nav** — 轻量、高效的 macOS 原生 Redis 管理工具

### 1.2 产品定位

面向开发者和运维人员的 macOS 原生 Redis 可视化管理工具，以 Rust + Tauri 为技术底座，提供流畅的操作体验、现代化的界面设计，兼顾轻量与功能完整性。

### 1.3 目标用户

| 用户角色 | 典型场景 |
|---------|---------|
| 后端开发者 | 本地开发调试，查看/修改缓存数据，排查数据问题 |
| 运维工程师 | 线上 Redis 实例监控、慢查询分析、内存优化 |
| 技术负责人 | 多环境实例管理、团队配置共享 |
| 全栈/独立开发者 | 个人项目 Redis 日常管理 |

### 1.4 核心差异化

- **macOS 原生体验**：不是套壳网页，而是真正符合 Mac 审美的原生交互
- **极致性能**：Rust 后端处理百万级 Key 无压力，内存占用极低
- **智能操作**：内置 CLI 自动补全、树形结构浏览、批量操作向导
- **安全可靠**：SSH 隧道 / TLS 原生支持，连接配置本地加密存储

## 3. 技术架构

### 3.1 技术选型

| 层级 | 技术 | 选型理由 |
|------|------|---------|
| **后端** | Rust | 极致性能、内存安全、跨平台 |
| **前端** | Tauri (v2) + Vue 3 + TypeScript | 轻量 Webview、组件化开发、生态丰富 |
| **UI 框架** | TDesign Vue Next | 成熟的企业级组件库，设计规范统一 |
| **样式** | Tailwind CSS 3 | 原子化 CSS，快速构建一致性界面 |
| **Redis 驱动** | redis-rs | Rust 官方生态最成熟的 Redis 客户端 |
| **状态管理** | Pinia | Vue 3 官方推荐，TypeScript 友好 |
| **构建工具** | Vite 5 | 极速 HMR，Tauri 官方推荐 |
| **图表** | ECharts | 监控数据可视化 |
| **代码编辑** | CodeMirror 6 | CLI 终端 / Lua 编辑器的代码高亮 |

### 3.3 数据流

```
用户操作 → Vue 组件 → Pinia Action → Tauri invoke → Rust Handler
    ↑                                                       │
    │                                                       ▼
Vue 响应式更新 ← Pinia State ← Tauri Event ← Rust Response
```

### 3.4 安全设计

- **连接密码**：使用 macOS Keychain 存储，不明文落盘
- **通信加密**：TLS 连接 + SSH 隧道双重保障
- **本地配置**：敏感字段 AES-256 加密存储
- **操作审计**：危险操作（FLUSHALL/FLUSHDB）二次确认 + 操作日志

## 4. 非功能需求

| 维度 | 指标 |
|------|------|
| **性能** | 启动时间 < 1s；百万 Key 列表加载 < 3s；内存占用 < 100MB |
| **稳定性** | 长时间连接自动重连；异常操作不崩溃；大 Key 读取分页 |
| **安全** | 敏感信息 Keychain 存储；TLS/SSH 加密传输；危险操作防护 |
| **易用** | 5 秒内完成首次连接；常用操作不超过 3 步；支持键盘快捷操作 |
| **兼容** | Redis 5.x ~ 7.x；macOS 12+；Standalone / Sentinel / Cluster |
