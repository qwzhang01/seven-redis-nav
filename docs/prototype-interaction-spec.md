# Seven Redis Nav — Mac 版 Redis 管理工具

## 交互说明与功能明细文档

---

## 一、产品概述

Seven Redis Nav 是一款仿 macOS 原生桌面应用风格的 Redis 可视化管理工具，以浏览器为载体提供完整的 Redis 运维与开发体验。整个界面采用 Mac 窗口隐喻设计——包含红绿灯按钮、标题栏、工具栏、侧边栏和状态栏——让用户在 Web 端也能获得接近原生客户端的操作体验。

### 技术架构

| 层级 | 技术 | 说明 |
|------|------|------|
| 页面结构 | HTML5 | 单页应用，入口为 `index.html` |
| 样式 | Tailwind CSS + 自定义 CSS | `style.css`（基础布局）+ `style-ext.css`（组件扩展） |
| 逻辑 | ESModule JavaScript | `app.js`（主入口）+ `components/` 目录下各组件模块 |
| 数据 | Mock 数据 | `data.js` 提供所有模拟数据 |
| 图标 | Remix Icon 4.2 | 通过 CDN 引入 |
| 图表 | Canvas 2D | 手写简易折线图绘制，无第三方图表库依赖 |

### 文件结构

```
/
├── index.html                  # 主页面入口
├── app.js                      # 主逻辑（状态管理、渲染调度、事件绑定）
├── data.js                     # Mock 数据模块
├── style.css                   # 基础样式（窗口、标题栏、工具栏、侧边栏、键面板等）
├── style-ext.css               # 扩展样式（各功能组件的详细样式）
└── components/
    ├── keyDetail.js            # 键详情面板组件
    ├── cli.js                  # 命令行终端组件
    ├── monitor.js              # 实时监控面板组件
    ├── slowlog.js              # 慢查询日志组件
    ├── pubsub.js               # 发布/订阅组件
    └── config.js               # 服务器配置面板组件
```

## 二、全局布局

页面采用 **Mac 窗口** 隐喻，整体结构自上而下为：

```
┌─────────────────────────────────────────────────────┐
│  标题栏（Titlebar） — 红绿灯 / 应用名 / 操作按钮    │
├─────────────────────────────────────────────────────┤
│  工具栏（Toolbar） — Tab切换 / 全局搜索 / 连接状态   │
├──────────┬──────────┬───────────────────────────────┤
│  侧边栏   │  键面板   │  详情/工作区面板              │
│  (240px)  │  (320px)  │  (自适应剩余宽度)             │
│  连接列表  │  键搜索   │  根据Tab显示不同内容：         │
│  数据库    │  键列表   │  浏览器/CLI/监控/慢查询/      │
│  用户信息  │  分页     │  发布订阅/配置                │
├──────────┴──────────┴───────────────────────────────┤
│  状态栏（Statusbar） — 服务器信息 / 性能指标          │
└─────────────────────────────────────────────────────┘
```

- 主体区域采用 CSS Grid 三栏布局：`grid-template-columns: 240px 320px 1fr`
- 窗口最大宽度 1400px，高度为 `calc(100vh - 80px)`，最小高度 720px
- 响应式：≤1100px 侧边栏缩窄；≤900px 改为单栏纵向排列

### 桌面壁纸背景

窗口外部以 `fixed` 定位的渐变色壁纸作为背景，使用多层径向渐变叠加，营造 macOS 桌面感：

- 左上粉红色 `#ff6b9d`
- 右上紫色 `#7c5cff`  
- 底部青色 `#00d4ff`
- 基底深蓝渐变 `#1a1a2e → #16213e → #0f3460`
